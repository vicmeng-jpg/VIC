# 學習進階 Markdown 語法：表格與影片 🚀

今天我來挑戰 Markdown 中比較進階的兩個功能：**建立表格**以及**嵌入影片**！

## 📊 1. 練習建立表格

Markdown 的表格是由 `|` (管線符號，通常在 Enter 鍵的上方) 和 `-` (減號) 所組成的。
這對整理資料或重點筆記非常實用！

以下是一個簡單的學習計畫表範例：

| 學習階段 | 目標與工具 | 完成狀態 |
| :--- | :--- | :---: |
| **階段一** | 建立 GitHub Pages 與環境設定 | ✅ 完成 |
| **階段二** | 熟悉基礎 Markdown 語法 (圖片、超連結等) | ✅ 完成 |
| **階段三** | 網站版面整理 (YAML 與目錄選單) | ✅ 完成 |
| **階段四** | 進階 Markdown (表格與嵌入多媒體) | 🐢 進行中 |

> 💡 **小秘訣**：
>
> 標題下方的 `| :--- |` 是用來控制對齊方向的：
> - `:---` 代表靠左對齊
> - `:---:` 代表置中對齊
> - `---:` 代表靠右對齊

## 🎬 2. 練習嵌入 YouTube 影片

在標準的 Markdown 中，並沒有專屬「嵌入影片」的語法。
但是！因為 Markdown 檔案可以完美相容 HTML 語法，所以如果我們想在網頁中放一部 YouTube 影片，我們可以直接使用 YouTube 提供的 **嵌入 (Embed)** 程式碼 (也就是 HTML 的 `<iframe>` 標籤)。
